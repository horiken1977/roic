🔧 技術的詳細

  • データソース: EDINET API v2（金融庁電子開示システム）
  • 対象期間: 2024年度（2024年4月〜2025年3月期）
  • ROIC計算式: NOPAT ÷ 投下資本
  • NOPAT: 営業利益 × (1 - 実効税率)
  • 投下資本: 総資産 - 現金及び現金同等物
